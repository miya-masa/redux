# React Router Redux TodoList
## ディレクトリ構成
* actions // 共通アクションクリエイター
* components // 共通コンポーネント
* containers // 共通コンテナー
* layouts // レイアウト要素
* reducers // 共通Reducer
* routes // 画面

        * components // todoコンポーネント
        * containers // todoコンテナー
        * reducers // todoReducer
* store // store
* styles // css等
